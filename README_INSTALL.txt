WASHCONTROL - Car Wash Management System
==========================================

QUICK START (Windows):
----------------------
1. Double-click INSTALL_AND_RUN.bat
2. Wait for installation (2-5 minutes)
3. Browser will open automatically
4. Login: admin
   Password: admin123

CREATE PORTABLE VERSION:
------------------------
1. Run INSTALL_AND_RUN.bat first
2. Double-click BUILD_PORTABLE.bat
3. Find portable version in: build/washcontrol_portable/
4. Copy entire folder anywhere and run RUN.bat

FEATURES:
---------
- Employee management
- Shift tracking
- Telegram/VK notifications
- Camera integration (TRASSIR/RTSP)
- AI summaries
- Excel reports
- Automatic backups
- Dark theme UI
- Works offline
- No internet required

SYSTEM REQUIREMENTS:
--------------------
- Windows 10/11
- 2GB RAM minimum
- 1GB free disk space
- No Python required (auto-installed)

DEFAULT CREDENTIALS:
--------------------
Login: admin
Password: admin123

IMPORTANT: Change password after first login!
