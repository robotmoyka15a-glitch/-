## 📦 КАК ЗАПУСТИТЬ

### Быстрый старт:
```bash
# Windows
ЗАПУСТИТЬ.bat

# Откройте браузер
http://localhost:8000
```

### Portable версия (рекомендуется):
1. Запустите `СОБРАТЬ_ПРИЛОЖЕНИЕ.bat`
2. Дождитесь сборки (2-5 мин)
3. Откройте `build/washcontrol_portable/`
4. Запустите `washcontrol.exe`

**Результат:**
- Один `.exe` файл
- Папка `data/` с базой данных
- Никакой установки не требуется!

## 💾 БЭКАПЫ

1. Настройки → вкладка "Бэкап"
2. "Создать бэкап сейчас" — ручное сохранение
3. "↻ Восстановить" — восстановление из копии
4. Автоматический бэкап ежедневно в 03:00

## 🔧 НАСТРОЙКИ

### Telegram:
1. Создайте бота через @BotFather
2. Получите токен и Chat ID
3. Настройки → Telegram → Сохранить

### AI:
1. Настройки → AI-интеграции
2. Выберите провайдера (Builtin/Ollama/CLO)
3. Проверьте кнопкой "Тест AI"
