# 🚿 WashControl

**Локальная система управления автомойкой для Windows**

Операционная платформа для автомойки: учёт сотрудников, журнал машин, отчёты, Telegram-бот, VK-уведомления, камеры TRASSIR и локальный AI-помощник.

---

## 📦 Быстрый старт (для не-технарей)

### Вариант 1 — Готовый инсталлятор (рекомендуется)

1. Скачайте файл `WashControl Setup 1.0.0.exe` из раздела [Releases](../../releases)
2. Запустите, нажмите **Установить**
3. Появится ярлык на рабочем столе — запустите WashControl
4. Войдите: логин `admin`, пароль `admin123`
5. **Сразу смените пароль** в разделе «Настройки → Пользователи»

### Вариант 2 — Запуск из исходников

Требования: [Python 3.11](https://python.org) + [Node.js LTS](https://nodejs.org)

```
1. Скачайте или клонируйте проект
2. Запустите: start_dev.bat
```

## ⚙️ Настройка после установки

### 1. Telegram-бот

1. Напишите [@BotFather](https://t.me/BotFather) → `/newbot`
2. Получите токен вида `123456789:ABCdef...`
3. Узнайте свой Chat ID: напишите боту [@userinfobot](https://t.me/userinfobot)
4. В приложении: **Настройки → Telegram** → вставьте токен и Chat ID

**Доступные команды бота:**
```
/status  — кто на смене прямо сейчас
/today   — итоги сегодняшнего дня
/cars 5  — последние 5 машин
/report  — полный отчёт за день
```
Также можно задать любой вопрос текстом — ответит AI или база данных.

### 2. VK уведомления

1. Перейдите: vk.com → Сообщества → Управление → API → Создать ключ
2. Выдайте права: `messages`, `wall`
3. В приложении: **Настройки → VK** → вставьте токен, ID владельца, ID группы

### 3. Камеры TRASSIR

В приложении: **Настройки → TRASSIR**
- IP-адрес сервера TRASSIR (например `192.168.1.100`)
- Порт: `8080` (HTTP)
- Логин/пароль от TRASSIR

После сохранения: **Камеры → Загрузить каналы**

### 4. AI-помощник (опционально)

Требует 4+ ГБ RAM. Работает на CPU, без GPU.

**Шаги:**
1. Скачайте модель: [Qwen2.5-3B-Instruct-GGUF](https://huggingface.co/Qwen/Qwen2.5-3B-Instruct-GGUF)
   - Файл: `qwen2.5-3b-instruct-q4_k_m.gguf` (~2.3 ГБ)
2. Поместите в папку `data/models/`
3. Установите зависимость:
   ```
   pip install llama-cpp-python --extra-index-url https://abetlen.github.io/llama-cpp-python/whl/cpu
   ```
4. В приложении: **Настройки → AI** → включите AI

> Если AI не установлен — система работает в полном объёме, просто отвечает на вопросы напрямую из базы данных.

## 🛡️ Безопасность

- JWT-авторизация с 12-часовыми токенами
- Хэширование паролей (SHA-256)
- Два уровня доступа: operator / admin
- Оператор видит только свои данные
- Все действия логируются в журнале событий
- Нет внешних обязательных облачных сервисов

## 🆘 Частые вопросы

**Q: Можно работать без интернета?**
A: Да, всё ядро работает локально. Telegram/VK не работают без интернета, но остальное — да.

**Q: Что будет если выключить ПК в середине смены?**
A: Смена останется открытой. При следующем включении всё продолжит работу. Данные не потеряются.

**Q: Как добавить второго оператора?**
A: Настройки → Пользователи → заполните форму → Создать.

**Q: Где хранится база данных?**
A: `data/washcontrol.db` — обычный SQLite-файл. Можно открыть любым SQLite-браузером.
