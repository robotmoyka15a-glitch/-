# 🎨 Дизайн-система WashControl

## ✅ Обновления дизайна (выполнено)

### 1. Тёмная цветовая гамма
Применена современная тёмная тема на основе палитры **Slate**:

| Переменная | Значение | Описание |
|------------|----------|----------|
| `--bg-primary` | `#0f172a` | Основной фон (Slate 900) |
| `--bg-secondary` | `#1e293b` | Фон карточек (Slate 800) |
| `--bg-tertiary` | `#334155` | Ховеры, границы (Slate 700) |
| `--text-primary` | `#f8fafc` | Основной текст (Slate 50) |
| `--text-secondary` | `#94a3b8` | Вторичный текст (Slate 400) |
| `--accent-primary` | `#3b82f6` | Акцент (Blue 500) |
| `--success` | `#10b981` | Успех (Emerald 500) |
| `--warning` | `#f59e0b` | Предупреждение (Amber 500) |
| `--danger` | `#ef4444` | Ошибка (Red 500) |

### 2. Логотип
Создан новый логотип с каплей воды:

**Файлы:**
- `/frontend/public/logo.svg` — основной логотип (SVG)
- `/frontend/src/components/RobotLogo.jsx` — React-компонент для интерфейса

**Описание логотипа:**
- Синяя капля воды с градиентом
- Блик и пузырьки для объёма
- Текст "WashControl" с акцентом на "Control"
- Подпись "СИСТЕМА УПРАВЛЕНИЯ АВТОМОЙКОЙ"

### 3. Шрифты
Подключён профессиональный шрифт **Inter** от Google Fonts:
- Отличная читаемость в любом размере
- Оптимизирован для экранов
- Поддержка кириллицы

### 4. UI-элементы
- **Кастомные скроллбары** — тонкие, в стиле темы
- **Выделение текста** — синее, с белым текстом
- **Фокус** — синяя обводка для доступности
- **Плавные переходы** — 0.15s для всех интерактивных элементов
