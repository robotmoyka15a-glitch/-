Частые вопросы

Можно ли работать без интернета?

Да! Всё ядро системы работает локально. Telegram/VK не работают без интернета, но основной функционал доступен полностью.

Что будет если выключить ПК в середине смены?

Смена останется открытой. При следующем включении всё продолжит работу. Данные не потеряются благодаря SQLite базе.

Как добавить второго оператора?

Зайдите в раздел «Настройки → Пользователи», заполните форму и нажмите «Создать». Новый оператор сможет войти под своими данными.

Где хранится база данных?

В папке data/washcontrol.db — это обычный SQLite-файл. Можно открыть любым SQLite-браузером для резервного копирования.

Нужен ли GPU для AI-помощника?

Нет, AI работает на CPU. Требуется 4+ ГБ оперативной памяти. Модель Qwen2.5-3B-Instruct скачивается отдельно.
