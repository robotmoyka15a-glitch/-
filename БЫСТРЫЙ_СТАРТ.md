# 🚀 WASHCONTROL - БЫСТРЫЙ СТАРТ БЕЗ УСТАНОВКИ PYTHON

## ⚡ САМЫЙ ПРОСТОЙ СПОСОБ (ДЛЯ КОНЕЧНОГО ПОЛЬЗОВАТЕЛЯ)

### Вариант 1: Готовый EXE (рекомендуется)

Если вам прислали готовую папку **`WashControl_portable`**:

1. **Распакуйте** папку в любое место (Рабочий стол, Программы и т.д.)
2. **Запустите** файл `ЗАПУСТИТЬ.bat` или `WashControl.exe`
3. **Откройте браузер** по адресу: http://localhost:8000
4. **Войдите**: логин `admin`, пароль `admin123`

**ВСЁ!** Python устанавливать не нужно - всё уже встроено в EXE файл.

---

### Вариант 2: Если нужно собрать самостоятельно

#### Что потребуется:
- **Только для сборки**: Python (один раз)
- **Для использования**: НИЧЕГО! (Python будет встроен в EXE)

#### Инструкция по сборке:

1. **Установите Python** (только если нет):
   - Скачайте с https://www.python.org/downloads/
   - При установке отметьте ✅ "Add Python to PATH"
   - Займёт 2 минуты

2. **Запустите сборщик**:
   - Дважды кликните на `СОБРАТЬ_АВТОНОМНО.bat`
   - Подождите 3-5 минут
   - Скрипт сам установит все зависимости

3. **Найдите готовое приложение**:
   - Папка: `build/portable/`
   - Файл: `WashControl.exe`
   - Ярлык: `ЗАПУСТИТЬ.bat`

4. **Используйте**:
   - Скопируйте папку `portable` куда угодно
   - Запустите `ЗАПУСТИТЬ.bat`
   - Python больше не нужен!

## 📦 ЧТО ВНУТРИ ПОРТАТИВНОЙ ВЕРСИИ


## 🔧 КАК ЭТО РАБОТАЕТ

При сборке используется **PyInstaller**, который:
1. Берёт весь код на Python
2. Упаковывает встроенный Python Interpreter
3. Добавляет все библиотеки (FastAPI, SQLAlchemy, bcrypt и др.)
4. Создаёт ОДИН .exe файл

**Результат**: Приложение работает БЕЗ установленного Python!

## ❓ ЧАСТЫЕ ВОПРОСЫ

### Нужно ли устанавливать Python для работы?
**НЕТ!** Python нужен только для сборки. Готовый EXE работает сам по себе.

### Можно ли перемещать папку?
**ДА!** Портативная версия работает из любой папки.

### Работает ли без интернета?
**ДА!** Все данные хранятся локально. Интернет нужен только для:
- Отправки уведомлений в Telegram/VK
- Синхронизации с Google Sheets
- Обновления AI-моделей (если используете облачные)

### Сколько ресурсов нужно?
- **RAM**: 150-300 МБ в простое
- **CPU**: Минимальная нагрузка (1-5%)
- **Место**: ~200 МБ + база данных

### Как остановить приложение?
Закройте консольное окно или нажмите `Ctrl+C` в консоли.

### Где база данных?
В папке `data/washcontrol.db`. Можно скопировать для бэкапа.

## 🆘 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Проблема: "Не удаётся запустить приложение"
**Решение**: 
- Проверьте, не блокирует ли антивирус
- Запустите от имени администратора
- Проверьте порт 8000 (не занят ли другой программой)

### Проблема: "Браузер не открывается"
**Решение**:
- Откройте вручную: http://localhost:8000
- Проверьте, работает ли приложение (должна быть открыта консоль)

### Проблема: "Неверный логин/пароль"
**Решение**:
- По умолчанию: `admin` / `admin123`
- Если меняли и забыли - см. раздел "Сброс пароля" в документации

### Проблема: "Ошибка базы данных"
**Решение**:
- Проверьте, есть ли права на запись в папку `data/`
- Восстановите из бэкапа через веб-интерфейс
