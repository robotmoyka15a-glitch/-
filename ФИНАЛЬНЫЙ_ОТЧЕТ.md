## 🚀 СЛЕДУЮЩИЕ ШАГИ

### Для запуска на автомойке:
1. ✅ Соберите приложение: `СОБРАТЬ_ПРИЛОЖЕНИЕ.bat`
2. ✅ Запустите: `build/washcontrol_portable/ЗАПУСТИТЬ.bat`
3. ✅ Войдите как `admin/admin123`
4. ✅ Смените пароль администратора
5. ✅ Создайте сотрудников
6. ✅ Настройте Telegram (@BotFather)
7. ✅ Протестируйте смену
8. ✅ Запустите в работу

### Для создания installer:
1. Установите NSIS (https://nsis.sourceforge.io/)
2. Запустите: `makensis.exe installer.nsi`
3. Получите `washcontrol_installer.exe`
4. Запустите installer на целевом ПК
