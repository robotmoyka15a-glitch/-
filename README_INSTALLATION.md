## 📋 ПОДРОБНАЯ ИНСТРУКЦИЯ

### Требования

| Что | Зачем | Где взять |
|-----|-------|-----------|
| Windows 10/11 | ОС для сервера | — |
| Python 3.10+ | Backend | https://python.org |
| Node.js 18+ | Frontend сборка | https://nodejs.org |
| 2 GB RAM | Минимум для работы | — |
| 500 MB HDD | Место для установки | — |

### Установка Python

1. Скачайте с https://python.org/downloads/
2. Запустите установщик
3. ⚠️ **ОБЯЗАТЕЛЬНО** отметьте галочку **"Add Python to PATH"**
4. Нажмите "Install Now"
5. Перезапустите командную строку
6. Проверьте: `python --version`

### Установка Node.js

1. Скачайте LTS версию с https://nodejs.org/
2. Запустите установщик
3. Нажмите "Next" → "Next" → "Install"
4. Перезапустите командную строку
5. Проверьте: `node --version`

## 💾 РЕЗЕРВНОЕ КОПИРОВАНИЕ

### Автоматические бэкапы

Система автоматически создаёт бэкапы:
- Ежедневно в 03:00

3. Кнопка "Восстановить"

## ❓ ДИАГНОСТИКА ПРОБЛЕМ

### Приложение не запускается

1. Проверьте логи: `data/logs/error.log`
2. Запустите `check_ready.py`
3. Убедитесь, что порт 8000 свободен

### Порт 8000 занят

```cmd
netstat -ano | findstr :8000
taskkill /PID <номер_процесса> /F
```

Или измените порт в настройках.

### Ошибки базы данных

1. Закройте приложение
2. Скопируйте `data/washcontrol.db` как резерв
3. Попробуйте восстановить из бэкапа

### Антивирус блокирует

1. Добавьте папку приложения в исключения антивируса
2. Это ложное срабатывание (self-signed сертификат)
